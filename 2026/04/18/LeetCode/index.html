Python常用数据结构与算法

基本数据结构

list - 列表

函数/方法 作用 时间复杂度 刷题场景举例
len(lst) 获取列表长度 滑动窗口（判断窗口大小）、数组遍历
lst.append(x) 尾部添加元素 结果收集（如滑动窗口最大值存入ans）
lst.pop() 尾部弹出元素 栈/队列模拟（默认弹最后，pop(0) 是 ）
lst.pop(i) 弹出指定下标元素 单调队列（移除窗口外元素）
lst.sort() 原地排序（Timsort） 数组排序、TopK问题预处理
lst.reverse() 原地反转列表 回文判断、反转数组
lst.index(x) 查找元素x的第一个下标（无则报错） 找元素位置、子数组匹配
lst.count(x) 统计元素x出现次数 频次统计、多数元素问题
lst[:] 列表切片（浅拷贝） 子数组/子串提取（如最小覆盖子串结果）
lst.extend(iter) 批量添加可迭代对象元素 合并多个列表

str - 字符串

函数/方法 作用 刷题场景举例
len(s) 获取字符串长度 字符串遍历、子串长度判断
s[i] 按下标访问字符 双指针（回文、滑动窗口）
s.split(sep) 按分隔符分割字符串 解析字符串格式（如”1,2,3”转列表）
s.join(iter) 拼接可迭代对象为字符串 结果拼接（如列表转字符串返回）
s.strip() 去除首尾空白/指定字符 字符串预处理（如去除无用空格）
s.replace(old, new) 替换子串 字符串修改、模式匹配
s.find(sub) 查找子串下标（无则返回-1） 子串存在性判断、滑动窗口匹配
s.isdigit() 判断是否全为数字 数值字符串校验
s.lower()/s.upper() 大小写转换 忽略大小写的字符串比较

dict - 字典

注意边界：如dict.get(key, 0)避免KeyError。

函数/方法 作用 刷题场景举例
d[key] 访问/赋值键对应的值（无则报错） 频次统计、两数之和（存值-下标映射）
d.get(key, default) 访问键值（无则返回default） 避免KeyError，频次统计（默认0）
d.keys() 获取所有键 遍历字典键（如判断字符是否在t中）
d.values() 获取所有值 频次统计后取值
d.items() 获取键值对（(k,v)） 遍历字典（如排序字典、筛选频次）
d.pop(key) 删除键并返回值 移除无用的键值对
del d[key] 删除键值对 同pop，无返回值

set - 集合

函数/方法 作用 刷题场景举例
set() 创建集合，常用于去重 数组/字符串去重 (list(set(nums)))
len(s) 获取集合长度 统计不重复元素个数
s.add(x) 添加元素到集合 记忆化搜索、哈希表模拟 (visited.add(node))
s.remove(x) 从集合移除元素（不存在则报错） 滑动窗口收缩（移除左边界元素）
s.discard(x) 从集合移除元素（不存在不报错） 安全地移除元素，避免 KeyError
x in s 检查元素是否在集合中 快速查找/存在性判断 (if node in visited:)

核心用法 刷题场景举例

deque - 双端队列

注意时空复杂度：比如列表pop(0)是，刷题时优先用的deque.popleft()。

函数/方法 作用 时间复杂度 刷题场景举例
q.append(x) 尾部添加元素 队列/栈模拟
q.appendleft(x) 头部添加元素 双端队列专用（如单调队列）
q.pop() 尾部弹出元素 栈模拟（后进先出）
q.popleft() 头部弹出元素 队列模拟、滑动窗口（单调队列移除左边界）
q[0] 访问队首元素 单调队列取窗口最大值
q[-1] 访问队尾元素 单调队列判断尾部元素是否需要弹出

Counter - 频次统计

函数/方法 作用 刷题场景举例
cnt[x] 获取/修改元素x的频次 同defaultdict(int)，更简洁
cnt.most_common(k) 获取前k个高频元素（列表，(元素,频次)） TopK问题（如前k个高频数字）
cnt1 == cnt2 判断频次是否完全一致 异位词判断（如有效的字母异位词）
cnt1.subtract(cnt2) 频次相减 最小覆盖子串（判断窗口是否满足）

heapq 库 - 堆/优先队列

Python 堆默认是小根堆，大根堆需通过“值取负”实现，是 TopK、滑动窗口最大值等问题的核心工具。

函数/方法 作用 时间复杂度 刷题场景举例
heapq.heappush(heap, x) 入堆（维护堆结构） 堆排序、滑动窗口最大值（存-值+下标）
heapq.heappop(heap) 出堆（弹出堆顶，最小元素） 移除堆顶无效元素（如窗口外的最大值）
heap[0] 访问堆顶元素（不弹出） 获取当前堆的最小值/最大值（取负后）
heapq.heapify(lst) 原地将列表转为堆 初始化堆（如滑动窗口前k个元素建堆）
heapq.nlargest(k, lst) 获取列表中前k个最大元素 TopK问题（比排序更高效）
heapq.nsmallest(k, lst) 获取列表中前k个最小元素 最小k个数问题

函数 作用 刷题场景举例
math.abs(x) 绝对值 数值比较、距离计算
math.sqrt(x) 平方根 几何题、距离公式（如欧氏距离）
math.pow(x, y) x的y次方 数值计算、幂次相关问题
math.gcd(a, b) 最大公约数 数论问题、分数化简
math.lcm(a, b) 最小公倍数 数论问题
math.inf 无穷大 初始化极值（如最小覆盖子串初始长度）
math.floor(x) 向下取整 数值处理、分桶问题
math.ceil(x) 向上取整 数值处理、需要整数结果的计算

bisect 库 - 二分查找

bisect仅适用于有序数组。

函数 作用 刷题场景举例
bisect.bisect_left(lst, x) 找x插入的最左位置（无则返回len(lst)） 二分查找、有序数组去重、找边界
bisect.bisect_right(lst, x) 找x插入的最右位置 统计x出现次数（right - left）
bisect.insort_left(lst, x) 插入x并保持有序 维护有序列表（如数据流中位数）

sortedcontainers 库 - 有序列表

类别 方法 语法 描述 示例
创建与初始化 构造函数 SortedList(iterable) 创建一个新的有序列表。如果传入可迭代对象，则将其元素排序后放入列表。 sl = SortedList([3, 1, 4]) 结果为 [1, 3, 4]
添加元素 add() sl.add(value) 将一个元素 value 添加到列表中，并保持列表的升序状态。 sl = SortedList([1, 3]); sl.add(2) 结果为 [1, 2, 3]
update() sl.update(iterable) 将一个可迭代对象中的所有元素添加到列表中，并保持列表的升序状态。 sl = SortedList([1]); sl.update([3, 2]) 结果为 [1, 2, 3]
删除元素 remove() sl.remove(value) 删除列表中第一个等于 value 的元素。如果不存在则会引发 ValueError。 sl = SortedList([1, 2, 3]); sl.remove(2) 结果为 [1, 3]
discard() sl.discard(value) 删除列表中第一个等于 value 的元素。如果不存在也不会引发错误。 sl = SortedList([1, 3]); sl.discard(2) (无错误，列表不变)
pop() sl.pop(index) 移除并返回指定索引 index 处的元素。默认为 -1 (最后一个元素)。 sl = SortedList([1, 2, 3]); val = sl.pop() val 为 3
clear() sl.clear() 清空列表，使其为空。 sl = SortedList([1, 2]); sl.clear() 结果为 []

位运算/异或

运算符 名称 符号 核心规则（逐位运算） 示例（十进制→二进制） 常用场景
左移 Shift Left << 所有位左移 n 位，右侧补 0，等价于 ×2ⁿ 3<<2 → 12（011→1100） 快速计算 2 的幂、生成定位符
右移 Shift Right >> 所有位右移 n 位，左侧补符号位，等价于 ÷2ⁿ（向下取整） 12>>2 → 3（1100→0011） 快速整除 2
按位与 AND & 对应位都为 1 则为 1，否则为 0 5&3 → 1（101&011=001） 判断某一位是否为 1、掩码筛选
按位或 OR \ 对应位有 1 则为 1，否则为 0 5\ 3 → 7（101\ 011=111） 置位（将某一位设为 1）
按位异或 XOR ^ 对应位不同则为 1，相同则为 0 5^3 → 6（101^011=110） 找不同、交换变量、消去重复数
按位取反 NOT ~ 所有位取反（Python 是～x = -(x+1)） ~5 → -6（101→…11111010） 快速取反、生成反掩码

可变对象/不可变对象

类型 定义 举例
不可变对象 创建后自身值无法修改，若要 “修改”，本质是创建新对象并让变量指向新地址 int、str、float、tuple、bool
可变对象 创建后自身值可以修改，修改时不会创建新对象，内存地址保持不变 list、dict、set、自定义类实例

不可变对象的「值」一旦创建就无法修改，所谓 “修改” 其实是变量指向了新的对象；

元组是特殊的不可变对象：若嵌套了可变对象（如列表），可变子对象的值可改，但元组本身的元素（子对象的引用）不可改。

直接赋值/浅拷贝/深拷贝

不可变对象赋值后指向原地址，但赋值后变量修改 = 创建新对象，不影响原变量。

所以以下我们讨论的是不可变对象的问题。

类型 定义 特点
直接赋值 仅传递对象的「引用」（相当于给对象起了一个新名字） 原对象和新变量指向同一个内存地址，修改一个会影响另一个
浅拷贝 创建新对象，但新对象的「子元素」仍引用原对象的子元素（只拷贝第一层） 修改新对象的「第一层元素」不影响原对象；修改「子元素」会影响原对象
深拷贝 创建新对象，且递归拷贝所有子元素（所有层级都拷贝） 原对象和新对象完全独立，修改任何层级的元素都互不影响
